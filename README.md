# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1jvD5r9_Mnfu4BGFdLsoArrLezSSBJQ6h

## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. Set your Gemini API key in the `.env` file:
   - Open `.env` and replace `YOUR_ACTUAL_API_KEY_HERE` with your actual API key
   - Get your API key from: https://aistudio.google.com/app/apikey
3. Run the app:
   `npm run dev`

The app will automatically use the API key from the `.env` file - no manual input required!
